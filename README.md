# git

Git alias plugin for [Oh My Fish][omf-link] - cloned from oh-my-zsh [plugins/git][omz-p-git-link].

## Install

```fish
$ omf install git
```


## Usage

See the [Source](git.fish) for a list of aliases and functions provided by the plugin.
